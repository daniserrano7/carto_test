-- OPTIONAL --
This can be used on feature click to get
entitiy's data instead of including all
data in geojson. On one hand, data size is
reduced and only the needed data is loaded.
On the other hand, every click has a delay 
due to latency. Considering geom layer 
has only ~100 features, all data can be included 
in geojson without impacting performance
